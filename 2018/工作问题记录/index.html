6.mvn clean install 找不到symbol问题

会报错maven-compile-plugin的版本问题，实际上是找不到依赖的包。这种问题可以从以下两个方面定位：

1.依赖错误，引用了低版本的jar包，部分本地仓库有低版本的拷贝，不会报错。

2.编译错误

有时候本地的引用类库和远程的类库不一致，或者低版本不被维护了，导致部分引用到低版本的在存在本地复制的情况下不会报错。
